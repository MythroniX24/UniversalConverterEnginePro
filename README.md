# Universal Converter Engine Pro v2.0 🚀

Full-featured Android file conversion app with AI-powered compression, 22+ features.

## 🎯 Features
- **Image**: PNG/JPG/WebP/BMP/GIF/TIFF/ICO/PDF conversion, Smart Compress AI, Compression Racing, SSIM Quality Score, Privacy Scanner, Metadata Nuker, 2x/4x Upscaler
- **Document**: Image→PDF, PDF→Image, Merge, Split, Compress, Watermark, Password Protect, PDF→Word
- **Media**: Audio extraction (AAC/MP3/WAV), Video→GIF, Video Compress, Subtitle Extractor
- **3D**: OBJ viewer (OpenGL ES 2.0), Wireframe mode, OBJ→STL export, Auto-rotate, Zoom
- **Dashboard**: Conversion history, Space saved stats, Success rate

## 🔧 Build
```bash
git clone https://github.com/USER/UniversalConverterEnginePro
cd UniversalConverterEnginePro
./gradlew assembleDebug
```
APK → `app/build/outputs/apk/debug/`

## 🚀 CI/CD
Push to main → GitHub Actions builds APK automatically → Download from Actions → Artifacts
